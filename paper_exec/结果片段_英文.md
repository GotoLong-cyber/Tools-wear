# 结果描述片段（原英文版文件，现统一为中文版）

## 主结果 / Table 2
在三折 LOCO 评估中，TimerXL head-only 的平均 MAE 为 8.7711 um，引入 TMA 后下降到 6.1484 um，而进一步引入完整的 KNN-DRR 后可下降到 3.1784 um。该结果说明，单靠 backbone 表征不足以消除跨工况任务中的结构性误差，而时间尺度增强与基于检索的残差修正分别在不同层级提供了互补收益。当前主结果来自 clean checkpoint 与 train-only 的 inner-LOSO 超参数选择协议。

## 代表性困难场景 / Fig.3 + Fig.5
代表性困难折上的主要误差集中在 late stage。head-only 模型的 late-stage MAE 达到 21.2180 um，而 delta-blend 将其降低到 5.7781 um。该结果表明，主导性失效模式不是整条曲线的均匀偏移，而是未来磨损增量的结构性低估，这也正是本文 retrieval-based residual correction 所针对的问题。

## TMA 机制 / Fig.6
TMA 的主要作用不是充当独立的最终预测提升模块，而是改善时间尺度变化下的 retrieval utility。在当前三折证据中，TMA 使 3/3 个 fold 的 late-stage delta error 都下降，说明训练期多步长增强能够为残差修正提供更有用的邻居。与此同时，原始 top-1 cosine distance 并未在所有折上单调下降，因此更准确的表述应是：TMA 改善的是 retrieval usefulness，而不是保证所有距离统计量都同步优化。
