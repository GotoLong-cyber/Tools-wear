# 结果描述片段（中文版）

## 主结果 / Table 2
在三折 LOCO 评估中，TimerXL head-only 的平均 MAE 为 8.7711 um，引入 TMA 后下降到 6.1484 um，而进一步引入 KNN-DRR 后可进一步下降到 3.1784 um。该结果说明单靠基础模型表征无法消除跨工况的结构性误差，而训练期时间尺度增强与基于检索的残差修正形成了清晰的两级修正链条。当前主结果来自 clean 训练 checkpoint 与 inner-LOSO 选出的固定检索参数，属于正式可用证据。

## 代表性困难场景 / Fig.3 + Fig.5
代表性困难折的核心问题集中在 late-stage。head-only 在 late-stage 的 MAE 达到 21.2180 um，而 delta-blend 可将其降低到 5.7781 um。该结果说明误差并非均匀分布在全曲线上，而是集中表现为未来磨损增量的系统性低估，因此 retrieval-based residual correction 具有明确的机制针对性。

## TMA 机制 / Fig.6
TMA 的主要作用不是直接替代预测头，而是改善检索在时间尺度变化下的可用性。从现有三折结果看，late-stage delta error 在 3/3 个 fold 上下降，说明训练期多步长增强有助于缓解跨工况时间尺度错位，并为 KNN-DRR 提供更有效的残差参考。与此同时，原始 top-1 cosine distance 并未在所有 fold 上单调下降（例如 fold1, fold2, fold3），因此更稳妥的表述应是：TMA 改善的是 retrieval utility，而不是保证所有距离指标同步优化。
