# 主结果部分草稿（中文）

## 5.1 主结果

表 2 汇总了本文在 PHM2010 三折留一工况评估下的主结果。整体来看，正式证据链呈现出清晰且稳定的三步提升关系：仅使用 TimerXL 骨干预测器时，三折平均 MAE 与 RMSE 分别为 `8.771060307820639` μm 和 `11.974687146777304` μm；在训练阶段引入 TMA 后，平均 MAE 与 RMSE 分别下降至 `6.148400783538818` μm 和 `9.462964462287232` μm；进一步在此基础上引入 KNN-DRR 后，平均 MAE 与 RMSE 进一步下降至 `3.1784137884775796` μm 和 `3.976317848759981` μm。该结果表明，单靠基础时序骨干网络的表征能力仍不足以消除跨工况条件下的结构性预测误差，而 TMA 与 KNN-DRR 在不同层级上提供了互补收益。

从折间结果看，这一提升并非仅集中在某一单折，而是在三折 LOCO 设置下均保持成立。对于 `c1,c4 -> c6` 的 fold1，TimerXL head-only、TimerXL + TMA 与 TimerXL + TMA + KNN-DRR 的 MAE 分别为 `13.398430824279785` μm、`9.276140213012695` μm 和 `3.2064666748046875` μm；对于 `c4,c6 -> c1` 的 fold2，相应结果分别为 `6.974664688110352` μm、`3.7426395416259766` μm 和 `3.3330025672912598` μm；对于 `c1,c6 -> c4` 的 fold3，相应结果分别为 `5.940085411071777` μm、`5.426422595977783` μm 和 `2.995772123336792` μm。与 TimerXL head-only 相比，完整方法在三折上均实现了显著下降，说明本文主线并非依赖单一工况或单一分割的偶然收益。

这一结果链也说明，TMA 与 KNN-DRR 解决的是不同层级的问题。TMA 的主要作用在于改善跨工况时间尺度变化下的表示可比性，因此它首先提升了骨干网络在 few-shot cross-condition 条件下的基础预测质量；但从数值上看，TMA 并未单独完成最终误差压缩，进一步决定整体性能上限的仍然是 KNN-DRR 对未来磨损增量残差的检索修正。换言之，TMA 负责改善“样本在表示空间中是否更可比”，而 KNN-DRR 负责修正“基础预测在未来增量上仍然低估了多少”。这两类作用并不重复，而是形成了由表示层增强到残差层纠偏的两级协同机制。

因此，本文的正式主结果不应被理解为“TimerXL 本身已解决跨工况预测问题”，而应理解为：TimerXL 提供稳定的骨干表示与基础预测，TMA 在训练阶段缓解时间尺度错位带来的表示失配，KNN-DRR 则在此基础上对未来磨损增量残差进行显式修正。正是在这一固定层级下，本文方法才在 clean protocol 与 inner LOSO train-only 选参协议下取得了稳定的三折提升。
