# TMA 消融实验结果

## 实验口径

- 特征：`Pure RMS7`
- 标签：`mean`
- 训练：`no validation set`
- 轮数：`epoch = 300`
- 外层评估：`LOCO`

## 三折总体 MAE

| 配置 | Fold1 MAE | Fold2 MAE | Fold3 MAE | 三折平均 MAE | 相对 Baseline |
|------|-----------:|-----------:|-----------:|--------------:|--------------:|
| Baseline | 4.4243 | 5.1470 | 4.3575 | 4.6429 | - |
| + TMA | 5.5669 | 4.9189 | 4.2480 | 4.9113 | -5.8% |
| + KNN | 2.2289 | 2.6133 | 2.5691 | 2.4704 | +46.8% |
| + TMA + KNN | 2.3226 | 2.7296 | 2.6175 | 2.5566 | +44.9% |

## 三折总体 RMSE

| 配置 | Fold1 RMSE | Fold2 RMSE | Fold3 RMSE | 三折平均 RMSE |
|------|------------:|------------:|------------:|--------------:|
| Baseline | 5.0665 | 6.2884 | 5.7984 | 5.7178 |
| + TMA | 5.9324 | 5.6726 | 5.2238 | 5.6096 |
| + KNN | 2.6342 | 2.9668 | 3.1348 | 2.9119 |
| + TMA + KNN | 2.6889 | 3.0900 | 3.2093 | 2.9961 |

## 结论

- `KNN` 是当前统一协议下的主要收益来源。
- 单独加入 `TMA` 并没有带来稳定提升，平均 MAE 反而从 `4.6429` 变为 `4.9113`。
- `Baseline + KNN` 的结果略优于 `TMA + KNN`：
  - `Baseline + KNN avg MAE = 2.4704`
  - `TMA + KNN avg MAE = 2.5566`
- 因此在当前最终协议下，`TMA` 不是必要模块，更合理的主线应为：
  - `Pure RMS7 + TimerXL + KNN`
