# 中文摘要（最终刷新版）

刀具磨损预测在 few-shot cross-condition 场景下尤其困难，因为不同工况下的磨损演化速率差异会导致固定时间步表示出现 temporal-scale misalignment，并在目标工况 late-stage 形成系统性的未来磨损增量低估。针对这一问题，本文以 TimerXL 作为时序 backbone，在其上引入两层互补机制：训练期多步长增强（TMA）用于提升跨工况表示可比性，基于检索的增量残差修正（KNN-DRR）作为主要创新，用于在推理阶段利用 train-only 检索库对未来磨损增量残差进行显式纠偏。当前正式结果均建立在 clean checkpoint 与 inner leave-one-source-out train-only 参数选择协议之上，outer test fold 仅评估一次。

在 PHM2010 数据集的三折 leave-one-condition-out 评估中，TimerXL head-only、TimerXL+TMA 和 TimerXL+TMA+KNN-DRR 的平均 MAE 分别为 `8.7711`、`6.1484` 和 `3.1784` um，对应平均 RMSE 分别为 `11.9747`、`9.4630` 和 `3.9763`。在代表性困难场景中，head-only 模型的 late-stage MAE 为 `21.2180` um，而引入 KNN-DRR 后可降至 `5.7781` um。结果表明，TMA 主要改善检索可用性，而 KNN-DRR 则直接针对跨工况下的 late-stage residual bias 提供稳定修正，从而形成清晰的性能提升链条。

**关键词**：刀具磨损预测；跨工况泛化；时序基础模型；检索增强预测；残差修正；小样本学习
