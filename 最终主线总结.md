# 最终主线总结

## 正式最优方案
`RMS7 + Feat_4 + A2-Random-S2 + SPECTRAL_ENTROPY_CH1 + Retrieval V2.1`

## Retrieval V2.1 定义
1. 主干模型固定为已训练好的 `Current-best`。
2. 在测试时从 `Timer-XL` 倒数第二层共享表示 `shared_repr` 建立检索与融合。
3. 使用 `delta retrieval`，目标不是直接投票未来绝对 wear，而是检索相似历史片段的未来增量。
4. 正式融合公式：
   - `delta_head = pred_head - current_last_wear`
   - `pred_final = pred_head + beta * (delta_knn - delta_head)`

## 正式推荐推理配置
1. 子库筛选：`q80`
2. 邻居数：`K = 5`
3. 距离：`cosine`
4. 融合权重：`beta = 0.5`
5. 融合模式：`delta_residual`
