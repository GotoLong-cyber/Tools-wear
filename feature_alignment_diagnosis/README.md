# 特征对齐诊断

该目录仅用于表征对齐诊断分析。
当前阶段不会在此目录直接启动训练任务。

## 目标范围

1. 诊断源域提取特征与目标域之间是否对齐。
2. 在每一折中保持严格的零泄露边界。
3. 产出可复现的可视化结果与定量对齐指标。

## 模块结构

1. `configs/`：按折、按域、按绘图配置的模板。
2. `modules/`：模块化实现规划（加载、标准化、可视化、指标、报告）。
3. `scripts/`：执行入口设计（待验收后实现）。
4. `outputs/`：图像与指标表的预期输出结构。

## 固定规则

1. `PCA` 作为主图，`UMAP` 作为辅助图。
2. 必须显式区分验证域与盲测域。
3. 磨损分段阈值仅可由训练域分位数拟合，再应用到 val/test。
4. 域分类 `AUC` 必须分开汇报：
   - 单特征 `AUC`
   - 整体特征集 `AUC`
5. `test_run` 仅用于事后诊断，不参与当折模型或特征决策。
