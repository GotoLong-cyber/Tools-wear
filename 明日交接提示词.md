本项目是刀具磨损跨工况预测，主模型是 TimerXL，当前主线是：

## 3. 当前阶段的关键事实（用于和导师对齐）
1. 双加载器问题已处理：采用“分别构窗 -> 合并 -> shuffle”训练机制。
2. 全局 tree top-k 路线验证后效果不理想（跨域泛化差）。
3. 已转为“RMS7基线 +1增广”路线：
   - `+Feat_4`（通道1 range）显著提升；
   - `+Feat_8`（通道2 range）在 hardest fold 退化；
4. 关键结论：
   - 不是“加特征越多越好”；
   - 需要选“能表征磨损且尽量不带域信息”的特征；
   - hardest fold 是主要决策依据。
5. 已建立对齐可视化流程（PCA/箱线图/域AUC/MMD/CORAL）。
6. 最新导师意见已落地为执行顺序：
   - 先做时间尺度证据化；
   - 再做切片策略对照；
   - 再做训练域 KNN 检索增强；
   - 最后再扩大特征组合验证。

## 4. 当前疑惑/待决策点
1. “只做 +1”是否会漏掉相关性组合特征？
   - 当前共识：先 +1 初筛，再做小规模组合补测。
2. 特征选择目标是“磨损表征”还是“工况表征”？
   - 当前共识：优先“磨损表征 + 跨域稳健”，不是纯工况识别。
3. 如何平衡对齐指标与真实 MAE？
   - 以三折 MAE（尤其 hardest fold）为主，诊断图用于解释。
4. 如何处理“单特征无效但组合可能有效”？
   - 当前共识：先 `+1` 过筛，再只对通过门槛的候选做 `+2`，不做全枚举。
